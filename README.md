
🌱 Energy & Sustainability Calculator
A realistic, data‑driven tool for estimating consumption and planning long‑term reductions.

This project is an interactive web application designed to help our institution estimate their energy, water, office supply and cleaning product consumption; and plan realistic reduction strategies over a 3‑year period.

The calculator integrates seasonal trends, holiday inactivity, variability, and custom reduction measures to generate accurate and meaningful sustainability insights.

Users can calculate:
- Next year’s approximate consumption
- Consumption for a selected period (using a custom month picker)
Units adapt automatically to the selected category (kWh, L, units/€).

Category‑Specific Reduction Planning
Each category includes a set of suggested reduction measures, each with a different impact percentage.
The user can:
- Select any combination of measures
- Apply them to the calculated consumption
- View the new reduced consumption instantly
- See the total percentage reduction achieved
